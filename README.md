# Crypto Tracker App

## Overview
A modern Android app built with Jetpack Compose for real-time cryptocurrency tracking, including charts, 24H price changes, and coin details.

## Features
- 📈 **Interactive Line Chart**
- 💰 **24H Price Change**
- 🔍 **Coin Details**
- 🌙 **Dynamic Theming**
- 🚀 **Modern Architecture**

## Tech Stack
- **Jetpack Compose**
- **Kotlin**
- **Canvas API**
- **KOIN** (DI)
- **Retrofit** (API calls)
- **ViewModel & StateFlow**
- **Coroutines & Flow**

## Setup
1. Clone the repo:
   ```sh
   git clone https://github.com/your-repo/crypto-tracker.git
   ```
2. Open in **Android Studio** and sync Gradle.
3. Add your crypto API key.
4. Run the app.
